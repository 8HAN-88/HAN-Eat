## 🚀 Быстрая настройка

### 1. Регистрация в ЮKassa

1. **Зарегистрируйтесь:**
   - 🔗 Регистрация: https://yookassa.ru/join
   - Заполните данные компании/ИП

2. **Получите ключи:**
   - 🔗 Личный кабинет: https://yookassa.ru/my
   - Перейдите в **Настройки** → **API**
   - Скопируйте:
     - **Shop ID** (идентификатор магазина)
     - **Secret Key** (секретный ключ)

### 2. Настройка backend/.env

Добавьте в `backend/.env`:

```env
# ЮKassa
YOOKASSA_ENABLED=true
YOOKASSA_SHOP_ID=ваш_shop_id
YOOKASSA_SECRET_KEY=ваш_secret_key
```

### 3. Установка зависимостей

```bash
cd backend
pip install yookassa==3.0.0
```

Или установите все зависимости:
```bash
pip install -r requirements.txt
```

### 4. Применение миграции

```bash
cd backend
alembic upgrade head
```

Это добавит поле `country_code` в таблицу `users`.

### 5. Настройка webhook

1. В личном кабинете ЮKassa перейдите в **Настройки** → **Уведомления**
2. Добавьте webhook URL:
   ```
   https://your-backend-url.com/api/v1/payments/webhook/yookassa
   ```
3. Выберите события:
   - `payment.succeeded` - успешная оплата
   - `payment.canceled` - отмена оплаты

## 🌍 Определение страны

Страна определяется автоматически:
1. Из профиля пользователя (`country_code`)
2. По языку браузера (`Accept-Language`)
3. По умолчанию: `RU` (Россия)

### Поддерживаемые страны для оплаты:

- ✅ **Россия (RU)** - ЮKassa (СБП, карты)
- ✅ **Беларусь (BY)** - ЮKassa
- ✅ **Казахстан (KZ)** - ЮKassa
- ❌ **Другие страны** - пока не поддерживается (можно добавить Stripe позже)

## 🔧 Как это работает

### Для российских пользователей:

1. Пользователь открывает экран подписки
2. Система определяет страну (RU)
3. Показываются цены в рублях
4. При нажатии "Купить" создается платеж через ЮKassa
5. Пользователь перенаправляется на страницу оплаты ЮKassa
6. Может выбрать: СБП, карту, электронный кошелек
7. После оплаты webhook создает подписку в БД

### Для других стран:

- Пока показывается сообщение, что оплата недоступна
- В будущем можно добавить Stripe для других стран

## 📝 API Endpoints

### POST `/api/v1/payments/checkout`

Создает платеж через ЮKassa (для России) или Stripe (для других стран).

**Request:**
```json
{
  "plan": "monthly" | "yearly"
}
```

**Response (для России):**
```json
{
  "payment_id": "2c8c4c4c-0001-5000-8000-000000000000",
  "url": "https://yoomoney.ru/checkout/payments/v2/...",
  "customer_email": "user@example.com",
  "provider": "yookassa",
  "currency": "RUB"
}
```

### POST `/api/v1/payments/webhook/yookassa`

Webhook для обработки событий от ЮKassa.

### GET `/api/v1/payments/prices`

Получить цены подписок (автоматически в зависимости от страны).
