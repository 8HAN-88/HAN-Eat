## 🔧 Что ВАМ нужно сделать вручную:

### 1. Создать файл `.env` (ОБЯЗАТЕЛЬНО)

Создайте файл `.env` в корне проекта (`D:\HAN Eat 1\.env`) со следующим содержимым:

```
SPOONACULAR_API_KEY=ваш_api_ключ_здесь
```

**Как получить API ключ:**
- Зайдите на https://spoonacular.com/food-api/console
- Зарегистрируйтесь или войдите
- Создайте новый API ключ
- Скопируйте ключ в файл `.env`

### 2. Настроить Firebase (ОБЯЗАТЕЛЬНО для полной функциональности)

#### Для Android:
1. Создайте проект в [Firebase Console](https://console.firebase.google.com/)
2. Добавьте Android приложение с package name: `com.example.han_eat`
3. Скачайте `google-services.json`
4. Скопируйте его в `android/app/google-services.json`

#### Для iOS:
1. В Firebase Console добавьте iOS приложение
2. Скачайте `GoogleService-Info.plist`
3. Скопируйте его в `ios/Runner/GoogleService-Info.plist`

### 3. Запустить бэкенд сервер (ОБЯЗАТЕЛЬНО для работы API)

```bash
cd RecipeApp
python app.py
```

Или если есть проблемы:
```bash
cd RecipeApp
python app_simple.py
```

### 4. Установить зависимости (если еще не сделано)

```bash
flutter pub get
```
