# Архитектура ленты постов (по мотивам ВКонтакте)

## Реализовано

### 1. Модель данных (`lib/models/post.dart`)

- **Post** - основная модель поста с поддержкой:
  - Разных типов: текст, фото, галерея, видео, опрос, ссылка, репост
  - Статусов: черновик, опубликован, на модерации, отклонен, удален
  - Реакций: лайки, комментарии, репосты, просмотры
  - Ранжирования: скоры релевантности, свежести, финальный скор
  - Бизнес-логики: промо-посты, реклама
  - Разнообразия: diversity groups для предотвращения дублирования

### 2. Сервис ленты (`lib/services/feed_service.dart`)

**Кандидаты контента:**
- Посты от друзей/подписок (высокий приоритет)
- Свежие посты (хронологические)
- Рекомендованные посты (алгоритмические)
- Рекламные посты (каждый 10-й)

**Ранжирование:**
- User Affinity (взаимодействия с автором)
- Content Relevance (релевантность контента)
- Recency Boost (экспоненциальный затухающий boost)
- Social Propagation (вовлеченность)
- Business Boost (для рекламы)
- Diversity Penalty (предотвращение дублирования)

**Формула ранжирования:**
```
final_score = 0.3 * user_affinity
            + 0.25 * content_relevance
            + 0.2 * recency_boost
            + 0.15 * social_propagation
            + 0.1 * business_boost
```

### 3. Экран ленты (`lib/features/feed/presentation/feed_screen.dart`)

- Переключатель режимов: "Для вас" / "Свежие"
- Бесконечная прокрутка (пагинация)
- Pull-to-refresh
- Обработка ошибок
- Пустые состояния

### 4. Карточка поста (`lib/features/feed/presentation/post_card.dart`)

Поддержка всех типов контента:
- Текстовые посты
- Фото (одно или галерея)
- Видео с превью
- Ссылки с превью
- Опросы
- Репосты (вложенные посты)

Действия:
- Лайк/анлайк
- Комментирование
- Репост
- Скрыть пост
- Пожаловаться
- Сохранить в закладки
- Скопировать ссылку

## Архитектура ранжирования

### Сигналы ранжирования

1. **Пользовательские сигналы**
   - История взаимодействий с автором
   - Частота взаимодействий
   - Настройки пользователя

2. **Контентные сигналы**
   - Тип поста (текст/фото/видео)
   - Наличие медиа
   - Длина текста
   - Теги

3. **Социальные сигналы**
   - Лайки, комментарии, репосты
   - Скорость роста вовлеченности
   - Взаимодействия друзей

4. **Временные сигналы**
   - Возраст поста
   - Экспоненциальное затухание

### Бизнес-правила

- Не более 1 рекламного поста на 10 обычных
- Не более 1 поста подряд от одного автора
- Разнообразие контента (diversity penalty)

## Использование

  path: '/feed',

)

  sortMode: FeedSortMode.personalized,
);
