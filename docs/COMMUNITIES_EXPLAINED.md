# Что такое Сообщества (Communities) в H.A.N. Eat?

## Концепция

**Сообщества** — это группы/каналы, похожие на:
- Группы в VK
- Каналы в Telegram
- Сообщества в Reddit

## Основные возможности

### 1. Создание сообщества
- Пользователь может создать сообщество (например, "Веганские рецепты", "Быстрые завтраки")
- Указывает название, описание, загружает обложку и аватар
- Назначает себя администратором

### 2. Страница сообщества
- Обложка и аватар
- Название и описание
- Счетчики: участники, посты
- Список постов сообщества
- Кнопка "Присоединиться"

### 3. Публикации от сообщества
- Администраторы могут публиковать посты **от имени сообщества**
- Посты автоматически появляются в лентах всех участников сообщества
- В посте видно, что он от сообщества (badge "От ЗОЖ Сообщества")

### 4. Участники
- Пользователи могут присоединяться к сообществам
- Администраторы могут назначать модераторов
- Участники видят посты сообщества в своей ленте

## Примеры использования

1. **"ЗОЖ Рецепты"** — сообщество о здоровом питании
   - Админ публикует рецепты с низкой калорийностью
   - Участники видят эти рецепты в ленте

2. **"Быстрые завтраки"** — сообщество для занятых людей
   - Рецепты, которые готовятся за 10-15 минут
   - Участники получают идеи для утренних блюд

3. **"Веганская кухня"** — сообщество для веганов
   - Только веганские рецепты
   - Обсуждения и обмен опытом

## Технически

### В базе данных уже есть:
- Таблица `communities` (создана в Phase 1)
- Таблица `community_members` (участники)
- Поле `community_id` в таблице `posts`

### Что нужно реализовать в Phase 3:
1. **Backend:**
   - POST /communities - создать сообщество
   - GET /communities/{id} - страница сообщества
   - POST /communities/{id}/join - присоединиться
   - POST /communities/{id}/posts - публикация от сообщества
   - GET /communities/{id}/posts - посты сообщества

2. **Frontend:**
   - Экран создания сообщества
   - Страница сообщества
   - Список сообществ
   - Публикация поста от имени сообщества
