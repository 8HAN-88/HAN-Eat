### 3. Типы постов в каналах

#### 3.1. Обычный пост
- ✅ Уже поддерживается через `type='text'` или `type='photo'`

#### 3.2. Короткое видео (рилсы)

#### 3.3. Рецепт (type='recipe')
- **КЛЮЧЕВАЯ ФУНКЦИЯ**
- При публикации рецепта в канале:
  1. Сохраняется в `posts` с `type='recipe'` и `channel_id`
  2. Автоматически добавляется в общую базу рецептов (Menu)
  3. Участвует в поиске по ингредиентам
  4. Отображается в ленте каналов
  5. Отображается в Menu

### 4. API эндпоинты

#### 4.1. Список каналов (обновить)
```
GET /api/v1/channels
  ?subscribed=true      # Мои каналы
  ?recommended=true     # Рекомендованные
  ?catalog=true         # Каталог
  ?category=...         # Фильтр по категории
  ?sort=popular|new     # Сортировка
  ?search=...           # Поиск
```

#### 4.2. Публикация рецепта в канале (новый)
```
POST /api/v1/channels/{channel_id}/recipe
Body: {
  title: string
  image_url: string
  ingredients: string[]  # или JSON
  steps: Step[]          # [{text, image_url?}]
  calories?: number
  cook_time?: number
  tags?: string[]
}
```

#### 4.3. Лента каналов (новый)
```
GET /api/v1/channels/feed
  ?limit=20
  ?offset=0
  ?channel_id=...       # Фильтр по каналу
```

### Связь с Menu

Рецепты из каналов должны:
1. Отображаться в Menu наравне с API рецептами
2. Участвовать в поиске по ингредиентам
3. Иметь те же фильтры (калории, время, категории)
4. Сохраняться в избранное
5. Участвовать в рекомендациях
