# Быстрое решение проблемы регистрации

## Проблема
Ошибка: "Backend сервер не запущен" при попытке регистрации

## Решение

### 1. Убедитесь, что backend сервер запущен

Проверьте в терминале, где запущен backend:
```bash
# Должны видеть что-то вроде:
# INFO:     Uvicorn running on http://0.0.0.0:5000
```

Если сервер не запущен, запустите его:
```bash
cd backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 5000
```

### 2. Проверьте, что сервер отвечает

Откройте в браузере: http://localhost:5000/docs

Должна открыться страница Swagger UI с документацией API.

### 3. Перезапустите Flutter приложение

После изменений в backend нужно перезапустить Flutter приложение:
- Остановите приложение (нажмите `q` в терминале Flutter)
- Запустите снова: `flutter run`

### 4. Проверьте настройки CORS

Я обновил настройки CORS в `backend/app/main.py` и `backend/app/core/config.py` для работы с Flutter web.

Если сервер был запущен с `--reload`, он должен автоматически перезагрузиться.

### 5. Если проблема сохраняется

Проверьте:
1. **Порт 5000 свободен**: `netstat -ano | findstr ":5000"`
2. **База данных запущена**: `docker-compose ps`
3. **Файл .env настроен**: проверьте `backend/.env`

### 6. Альтернативное решение

Если проблема в CORS, можно временно разрешить все origins в `backend/app/main.py`:
```python
app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],  # Разрешить все origins
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

## Проверка работы

После выполнения всех шагов попробуйте снова зарегистрироваться. Ошибка должна исчезнуть.
