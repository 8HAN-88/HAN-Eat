## Особенности реализации

### ModerationService
- Интеграция с backend API `/api/v1/moderation/pending`
- Поддержка фильтров и пагинации
- Методы для одобрения и отклонения

### Moderation Queue Screen
- ExpansionTile для каждого элемента с предпросмотром
- Цветовая индикация причины модерации
- Иконки для разных типов контента
- Диалоги для действий модератора
- Автоматическое обновление списка после действий

## Типы контента

1. **post** - Посты
2. **comment** - Комментарии
3. **user_profile** - Профили пользователей

## Причины модерации

1. **auto_flagged** - Автоматически помечено (AI)
2. **spam** - Спам
3. **inappropriate** - Неподходящий контент
4. **copyright** - Нарушение авторских прав
5. **other** - Другое
