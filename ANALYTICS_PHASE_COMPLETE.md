# Analytics Phase - Завершено ✅

## Выполнено

### 1. ✅ AnalyticsService
- Создан сервис для работы с аналитикой
- Методы для получения аналитики поста и профиля
- Поддержка параметра days для выбора периода
- Модели данных: PostAnalyticsResponse, ProfileAnalyticsResponse, DailyCount

### 2. ✅ Analytics Screen
- Создан экран аналитики для авторов
- Поддержка двух режимов: аналитика поста и аналитика профиля
- Селектор периода (7, 30, 90, 365 дней)
- Отображение основных метрик
- График просмотров по дням (простой line chart)
- Топ посты для профиля
- Метрики вовлеченности (CTR, Engagement Rate)

### 3. ✅ Маршруты
- Добавлен маршрут для Analytics Screen
- Поддержка параметра postId для аналитики конкретного поста

## Особенности реализации

### AnalyticsService
- Интеграция с backend API `/api/v1/analytics/posts/{id}` и `/api/v1/analytics/profile`
- Обработка ошибок
- Типизированные модели данных

### Analytics Screen
- Адаптивный UI с карточками метрик
- Простой line chart для визуализации просмотров
- Цветовая кодировка метрик
- Иконки для разных типов метрик
- Отображение топ постов для профиля

## Метрики

### Пост
- Просмотры (total, unique)
- Лайки
- Комментарии
- Сохранения
- Репосты
- CTR (Click-Through Rate)
- Engagement Rate
- График просмотров по дням

### Профиль
- Общая статистика (посты, каналы, подписчики)
- Метрики контента (просмотры, лайки, комментарии, сохранения, репосты)
- Средний Engagement Rate
- График просмотров по дням
- Топ посты
