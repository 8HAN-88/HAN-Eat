## 🔍 Диагностика:

### Шаг 1: Проверьте логи бэкенда

При запуске бэкенда должно быть:
```
✅ SPOONACULAR_API_KEY настроен (длина: 32)
```

При запросе к `/recommendations` должно быть:
```
🌐 Запрос к Spoonacular: https://api.spoonacular.com/recipes/random с параметрами {...}
📡 Ответ Spoonacular: status=200
📦 Ответ Spoonacular: keys=['recipes']
📋 Получено X рецептов из Spoonacular random API
🔍 Обработка рецепта 123: Recipe Title... image=True
✅ Recommendation 123: image=True, calories=350
     Image URL: https://spoonacular.com/recipeImages/...
📤 Возвращаем X рецептов с данными
```

### Шаг 2: Если видите "📋 Получено 0 рецептов"

**Возможные причины:**
1. Spoonacular API не возвращает рецепты (проверьте лимиты)
2. Неправильный формат ответа
3. Проблема с API ключом

**Что делать:**
- Проверьте логи - должно быть сообщение с содержимым ответа
- Проверьте, что API ключ активен
- Проверьте лимиты API на Spoonacular

### Шаг 3: Проверьте API напрямую

**Откройте:**
```
http://localhost:5000/docs
```

**Вызовите:**
- `GET /recommendations?limit=1`

**Проверьте ответ:**
- Должны быть поля `image` и `calories`
- `image` должен быть URL (не null и не пустая строка)
- `calories` должен быть числом (или null)
