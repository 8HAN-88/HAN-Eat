# 🔍 Отладка: Картинки и калории не отображаются

## 📋 Шаги для диагностики:

### 1. Проверьте, что бэкенд возвращает данные:

**Откройте в браузере:**
```
http://localhost:5000/docs
```

**Вызовите endpoint:**
- `GET /recommendations?limit=1`
- Проверьте ответ - должны быть поля `image` и `calories`

**Или через curl:**
```powershell
curl http://localhost:5000/recommendations?limit=1
```

### 2. Проверьте логи бэкенда:

В терминале, где запущен `python run.py`, должны быть сообщения:
```
📋 Получено X рецептов из Spoonacular
🔍 Обработка рецепта 123: Recipe Title... image=True
✅ Рецепт 123: image=True, calories=350, ingredients=5
```

### 3. Проверьте консоль Flutter:

В терминале Flutter должны быть сообщения о загрузке изображений.

### 4. Проверьте Network в DevTools:

1. Откройте DevTools (F12 в Chrome)
2. Перейдите на вкладку Network
3. Найдите запрос к `/recommendations`
4. Проверьте ответ - должны быть `image` и `calories`

## ⚠️ Возможные проблемы:

### Проблема 1: SPOONACULAR_API_KEY не настроен

**Решение:**
- Проверьте, что в `backend/.env` есть `SPOONACULAR_API_KEY=ваш_ключ`
- Перезапустите бэкенд

### Проблема 2: Spoonacular API не возвращает nutrition

**Решение:**
- Некоторые рецепты могут не иметь данных о питании
- Это нормально, калории будут `null` для таких рецептов

### Проблема 3: Изображения блокируются CORS

**Решение:**
- Spoonacular должен разрешать загрузку изображений
- Если проблема, можно добавить прокси на бэкенде

### Проблема 4: Данные не доходят до фронтенда

**Решение:**
- Проверьте Network в DevTools
- Проверьте, что бэкенд запущен на `http://localhost:5000`
- Проверьте CORS настройки
