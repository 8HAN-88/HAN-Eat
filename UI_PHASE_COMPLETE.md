## Особенности реализации

### Notifications Screen
- Использует NotificationService для загрузки уведомлений
- Поддержка пагинации (offset-based)
- Автоматическое обновление счетчика непрочитанных
- Визуальное выделение непрочитанных уведомлений
- Форматирование дат (относительное время)

### Support Screen
- Интеграция с SupportService
- Валидация полей формы
- Быстрое действие для отмены подписки
- Интеграция с SubscriptionService для запроса отмены

### Subscription Screen
- Проверка статуса подписки при загрузке
- Отображение даты истечения подписки
- Кнопка для запроса отмены через поддержку
- Переход на экран поддержки после создания запроса
