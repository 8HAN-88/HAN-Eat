1. ✅ **MediaService**
   - Генерация presigned URLs для загрузки в S3

## Особенности

1. **Presigned URLs** - безопасная загрузка напрямую в S3
2. **Mock режим** - работает без реального S3 для разработки
3. **Валидация** - проверка типов и размеров файлов
4. **Прогресс** - отображение прогресса загрузки
5. **Preview** - предпросмотр выбранных файлов перед публикацией
