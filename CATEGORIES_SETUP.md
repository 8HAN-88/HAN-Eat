# 🏷️ Инструкция по настройке Категорий рецептов

## ✅ Что реализовано

1. **Модели данных** (`lib/models/recipe_category.dart`):
   - `RecipeCategory` - 19 категорий (ЗОЖ, Дешево, Веган, Быстро, и т.д.)
   - `CategoryFilter` - фильтр с активностью и приоритетом
   - `CategoryType` - типы категорий (практические, диетические, тип приема пищи, кухни)

2. **Сервис** (`lib/services/category_service.dart`):
   - Локальное хранение через SharedPreferences
   - Управление активными категориями
   - Получение тегов для Spoonacular API

3. **UI экран** (`lib/features/categories/presentation/categories_screen.dart`):
   - Группировка категорий по типам
   - Переключатели для выбора категорий
   - Сброс всех фильтров

4. **Интеграция**:
   - Кнопка категорий в AppBar MenuScreen
   - Отображение активных категорий как чипсы
   - Фильтрация поиска и рекомендаций по категориям
   - Ссылка в настройках
   - Маршрут `/categories`

## 🔧 Настройка

### 1. Установи зависимости

```bash
flutter pub get
```

### 2. Сгенерируй Hive адаптеры (если используешь Hive для категорий)

```bash
flutter pub run build_runner build --delete-conflicting-outputs
```

### 3. Проверь инициализацию

Убедись что в `lib/app/bootstrap.dart` добавлено:
- Импорт `CategoryService`
- Инициализация `CategoryService.init()`

## 📱 Использование

1. **Выбрать категории**:
   - Нажми кнопку категорий в MenuScreen (иконка категорий)
   - Или зайди в Настройки → "Категории рецептов"
   - Выбери нужные категории переключателями

2. **Просмотреть активные категории**:
   - В MenuScreen активные категории отображаются как чипсы под строкой поиска
   - Можно удалить категорию, нажав на крестик в чипсе

3. **Фильтрация**:
   - При поиске рецептов будут учитываться активные категории
   - Рекомендации также фильтруются по активным категориям

## ⚠️ Важно

- Категории сохраняются локально через SharedPreferences
- Теги категорий передаются в Spoonacular API для фильтрации
- После изменения категорий автоматически обновляются рекомендации
- Для поиска по ингредиентам теги используются в эндпоинте `/recommendations`

## 🔄 Интеграция с API

Теги категорий передаются в запросы:
- `ApiService.fetchRecommendations()` - через параметр `tags`
- `ApiService.searchRecipes()` - через параметр `tags` (для будущей поддержки)

Сервер должен поддерживать параметр `tags` в эндпоинтах `/recommendations` и `/recipes`.
