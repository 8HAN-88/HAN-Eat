# 🚀 ЗАПУСК ПРЯМО СЕЙЧАС - Простая инструкция

## ⚡ Самый быстрый способ:

### Вариант 1: Автоматический запуск (рекомендуется)

**Дважды кликните на файл:**
- `start_all.bat` (для Windows)

Или в терминале:
```bash
.\start_all.bat
```

**Это запустит всё автоматически!** ✅

---

## 📋 Вариант 2: Ручной запуск (2 терминала)

### Терминал 1 - Бэкенд сервер:

```bash
cd "D:\HAN Eat 1\RecipeApp"
python app_simple.py
```

**⚠️ ВАЖНО:** Если появится ошибка про SPOONACULAR_API_KEY:
- Создайте файл `RecipeApp/.env` с содержимым:
  ```
  SPOONACULAR_API_KEY=test_key
  ```
- Или используйте `python app.py` (может работать без ключа)

**Оставьте это окно открытым!**

### Терминал 2 - Flutter приложение:

```bash
cd "D:\HAN Eat 1"
flutter run -d chrome
```

**Приложение откроется в браузере!** 🎉
