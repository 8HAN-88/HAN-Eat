## ⚠️ Что нужно сделать ВРУЧНУЮ

### 1. **Создать файл `.env`**
   - ❌ Код ожидает `.env` с `SPOONACULAR_API_KEY`
   - 🔧 **Решение:** Создать `.env` в корне проекта:
     ```
     SPOONACULAR_API_KEY=ваш_ключ_здесь
     ```
   - 📝 **Инструкция:** См. `SETUP_INSTRUCTIONS.md`

### 2. **Настроить Firebase**
   - ❌ Нужны файлы `google-services.json` (Android) и `GoogleService-Info.plist` (iOS)
   - 🔧 **Решение:** 
     - Создать проект в Firebase Console
     - Добавить Android/iOS приложения
     - Скачать конфигурационные файлы
     - Поместить их в соответствующие папки
   - 📝 **Инструкция:** См. `SETUP_INSTRUCTIONS.md` раздел 2

### 3. **Запустить бэкенд сервер**
   - ⚠️ `ApiService` теперь автоматически определяет URL, но сервер должен быть запущен
   - 🔧 **Решение:** 
     - Создать `RecipeApp/.env` с `SPOONACULAR_API_KEY`
     - Запустить `python RecipeApp/app.py` или `app_simple.py`
   - 📝 **Инструкция:** См. `SETUP_INSTRUCTIONS.md` раздел 3

### 4. **Для реальных устройств (опционально)**
   - ⚠️ Для реальных Android/iOS устройств нужно установить IP адрес компьютера
   - 🔧 **Решение:** 
     ```dart
     ApiService.setBaseUrl('http://192.168.1.XXX:5000'); // ваш IP
     ```
   - 📝 **Инструкция:** См. `SETUP_INSTRUCTIONS.md` раздел 3B
