## Как это работает

### Для российских пользователей:

1. Пользователь открывает экран подписки
2. Система определяет страну (RU) по языку браузера или профилю
3. Показываются цены в рублях:
   - Месячная: 299 ₽
   - Годовая: 2499 ₽
4. При нажатии "Купить" создается платеж через ЮKassa
5. Пользователь перенаправляется на страницу оплаты ЮKassa
6. Может выбрать способ оплаты:
   - ✅ **СБП** (Система быстрых платежей)
   - ✅ Банковские карты
   - ✅ Электронные кошельки
7. После оплаты webhook создает подписку в БД

### Для других стран:

- Пока показывается сообщение, что оплата недоступна
- В будущем можно легко добавить Stripe для других стран

## Что нужно сделать для запуска

### 1. Регистрация в ЮKassa

1. Зарегистрируйтесь: https://yookassa.ru/join
2. Получите ключи: https://yookassa.ru/my → Настройки → API
3. Скопируйте Shop ID и Secret Key

### 2. Настройка backend/.env

Добавьте в `backend/.env`:

```env
YOOKASSA_ENABLED=true
YOOKASSA_SHOP_ID=ваш_shop_id
YOOKASSA_SECRET_KEY=ваш_secret_key
```

### 3. Установка зависимостей

```bash
cd backend
pip install yookassa==3.0.0
```

### 4. Применение миграции

```bash
cd backend
alembic upgrade head
```

### 5. Настройка webhook

В личном кабинете ЮKassa добавьте webhook:
```
https://your-backend-url.com/api/v1/payments/webhook/yookassa
```
